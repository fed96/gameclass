О программе GameClass


GameClass


GameClass - программа для контроля и управления компьютерным клубом или интернет-кафе. Автоматизирует процесс работы компьютерного салона и защищает интересы его владельца, выполняя задачу постоянного контроля ресурсов. Большое количество функций базовой поставки обеспечивает удобство работы менеджера, операторов и посетителей салона. Программа привлекает низкой стоимостью. Функциональность программы можно расширить с помощью дополнительных модулей..


Основные возможности

  • Развитая система тарифов

  • Учет печати

  • Учет трафика (Windows/Linux/FreeBSD)

  • Учет состава оборудования

  • Многопользовательская среда

  • Гибкая система отчетов

  • Автоматическая инсталляция/обновление клиентских частей

  • Возможность удаленного мониторинга

  • Интеграция с RunPad Shell

  • Улучшенная блокировка клиентского компьютера

  • Система личных учетных записей

  • И многое другое..


    • Системные требования
  • Celeron 500 MHz или выше

  • 128 Mб ОЗУ или больше

  • 50Мб свободного дискового пространства

  • Windows 2000/XP/2003 (серверная часть)

  • Microsoft MSDE2000 (серверная часть)

  • Windows 2000/XP/2003 (клиентская часть)
