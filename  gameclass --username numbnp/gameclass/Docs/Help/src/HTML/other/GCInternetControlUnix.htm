GCInternetControl для Linux/FreeBSD

GCInternetControl for Linux/FreeBSD - модуль выполняет две функции: блокировку и качественный (с точностью до байта) подсчет трафика на Linux или FreeBSD, при этом нагрузка на Unix-сервер минимальна. Модуль собирает информацию о трафике и передает ее на GameClass Server, где и происходит подсчет итоговой стоимости сеанса пользователя. Модуль выполняет подсчет исходящего/входяшего трафика любого типа (TCP, UDP, ICMP), т.е. с помощью него можно учитывать не только работу в Internet Explorer, но и в играх по интернету, что открывает новые возможности для Вашего бизнеса. Модуль поставляется с подробным описанием и исходным кодом (необходима компиляция кода на вашем Linux/FreeBSD-Сервере).

Если же в качестве роутера Вашего интернет-салона используется UNIX-сервер (Linux или FreeBSD), а интернет тарифицируется по времени а не по трафику, то Вы можете использовать бесплатный модуль, выполняющий только блокировку трафика (без подсчета). Бесплатный модуль работает так: в зависимости от тарифа, по которому работает посетитель, приложение оператора отправляет команду модулю контроля интернет трафика, который с помощью команд файрволла блокирует либо разблокирует ВЕСЬ интернет-трафик. Загрузить бесплатный модуль можно здесь: http://nodasoft.com/download/gc/unix.on.off.zip (100 Кб). Для настройки бесплатного модуля в GameClass в настройках раздела "Трафик" нужно поставить галочку "Активировать блокирование трафика на UNIX-сервере" и указать АйПи адрес сервера в настройке "IP-адрес модуля подсчета трафика".
