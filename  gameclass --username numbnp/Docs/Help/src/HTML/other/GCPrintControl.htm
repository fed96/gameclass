Опциональные модули для GameClass

GCPrinterControl

GCPrinterControl - модуль контроля принтера представляет собой отдельное приложение PrintMonitor (http://www.printmonitorpro.com/download.html). В момент, когда принтеру было отправлено задание на распечатку и оно успешно распечаталось, модуль определяет, сколько страниц, когда и с какого компьютера было распечатано. Если печать выполнялась с клиентского компьютера, то из счета клиента, который выполнил распечатку, автоматически вычитается стоимость страниц. GameClass, PrintMonitor и принтеры могут находиться на разных компьютерах!

Покупая модуль, Вы бесплатно получаете лицензию на программу PrintMonitor и можете использовать его как совместно с GameClass, так и по отдельности.

Внимание! Модуль контроля принтера корректно работает только при печати с компьютеров с операционными системами Windows XP/2000/2003. При печати из под Windows 95/98/ME не отслеживается количество распечатанных страниц.

Определение количества копий напечатанного документа — одна из наиболее сложных задач при анализе данных о печати, т.к. функцию определения количества копий приходится писать отдельно для каждой новой линейки принтеров. Немногие программы учета печати могут похвастать таким функционалом. PrintMonitor начиная с версии 3.2 верно учитывает копии для наиболее распространенных моделей принтеров. Если на вашем принтере учет копий ведется неправильно, напишите письмо разработчикам. В очередной версии поддержка вашего принтера будет обязательно добавлена.
