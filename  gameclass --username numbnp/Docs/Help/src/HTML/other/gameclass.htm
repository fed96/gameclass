GameClass

GameClass - комплекс программного обеспечения для контроля и управления компьютерным клубом или интернет-кафе. GameClass автоматизирует процесс работы клуба и защищает интересы его владельца, выполняя задачу постоянного контроля ресурсов. Большое количество функций базовой поставки обеспечивает удобство работы менеджера, операторов и посетителей салона. GameClass привлекает низкой начальной стоимостью. При необходимости можно расширить функциональность комплекса с помощью дополнительных модулей, специально разработанных для продукта GameClass.

Реализованные возможности (в версии 3.7.2103.0)
  • Предоплата либо Постоплата
  • Многоязыковая поддержка
  • Настройка прав на функции системы
  • Почасовые и пакетные тарифы
  • Удаленное получение отчетов
  • Удаленный мониторинг
  • Эргономичное рабочее место оператора
  • Многопользовательская среда
  • Одновременная работа пользователей
  • Шаблоны для снятия задач и их заполнение в окне снятия задач
  • Возможность использования бесплатного тарифа "ремонт"
  • Управление FireWall unix-сервера (модуль контроля интернет-траффика)
  • Контроль состава оборудования (модуль контроля оборудования)
  • Контроль принтера (модуль контроля принтера)
  • Контроль интернет-трафика (модуль контроля интернет трафика)
  • Система клубных карт, личных счетов клиентов, накопительных скидок (в процессе тестирования)
  • Звуковые предупреждения клиентской части
  • Функция удаленного завершения процессов
  • Клиентская часть отображает номер компьютера, время, полную информацию по балансу
  • Возможность узнать текущую информацию по балансу с помощью иконки в SysTray (клиентская часть)
  • Возможность настроить тип блокировки для клиентской части
  • Перезагрузка, выключение клиентской части
  • Возможность синхронизации времени клиентов с сервером
  • В главном отчете подробная итоговая информация
  • Возможность ввести кол-во распечатанных страниц вручную
  • Возможность выбора любого кол-ва времени при заказе времени
  • Возможность автоматического снятия задач после завершения работы по истечение N минут
  • Возможность автоматической перезагрузки компьютера после завершения работы по истечение N минут
  • Настраиваемый звуковой сигнал при потери связи с компьютерами
  • Автоматическое подставление нужного интервала в главном отчете Функция контроля версий клиентских частей
  • Переименование функции "Ремонт на 1 минуту" в "Бесплатная минута"
  • Настройка кол-ва часов для функции "Ремонт надолго"
  • Удаленная деинсталляция клиентских частей (в режиме менеджера)
  • Просмотр состояния зала на ЛЮБОЙ ПРОШЕДШИЙ момент времени
  • и многое другое ....
