Работа с GameClass

Функции оператора


1. Запуск сеанса

Для начала сессии нужно выбрать свободный компьютер в списке компьютеров и вызвать окно "начала работы" одним из способов, двойной клик левой кнопкой мыши, клавиша Ins или через контекстное меню (правой кнопкой на выбранном компьютере).

Появится окно, в котором можно задать режим работы клиента.





2. Завершение сеанса

Для завершения сессии нужно выбрать занятый компьютер в списке компьютеров и вызвать окно "завершения сеанса" одним из способов: клавиша Del или через контекстное меню (правой кнопкой на выбранном компьютере).

Посмотреть, как выглядит окно завершения сеанса.





3. Продление сеанса

Если клиент хочет доплатить, то для внесения доплаты нужно выбрать компьютер клиента и вызвать окно "внесения доплаты" одним из способов: клавиша Ctrl+Ins или через контекстное меню (правой кнопкой на выбранном компьютере).

Посмотреть, как выглядит окно продления сеанса.





4. Перенос сеанса на другой компьютер

Если клиент хочет пересесть на другой компьютер или поменяться компьютерами с другим клиентом, то нужно вызвать функцию смены компьютера одним из следующих способов, клавиша Ctrl+M либо через контекстное меню (правой кнопкой мыши на выбранном компьютере).

Посмотреть, как выглядит окно переноса сеанса.





5. Использование тарифа "Ремонт"

Ремонт - это специальный тариф с нулевой стоимостью. Этот тариф должен применяться техническим работником для ремонта или настройки компьютеров. Возможны два варианта использования тарифа ремонт - "бесплатная минута" или "ремонт надолго". "Ремонт надолго" разблокирует компьютер на 360 минут (установлено по умолчанию, менеджер может выставить любое значение). Активация этого типа ремонта требует ввода комментария. Оба типа ремонта или один из них можно отключить, забрав права на соответствующую функцию у группы "Staff" (Операторы). В главном отчете отображается информация о применении тарифа ремонт и оставленные техническими работниками комментарии.

Посмотреть, как выглядит включение тарифа "ремонт".





6. Просмотр отчетов

Если оператору не запрещена возможность просматривать отчеты оператора, то он может видеть несколько отчетов: отчет по нескольким последним сеансам (позволяет восстановить картину работы если у посетителя возникли вопросы) и Отчет по смене, в котором представлена итоговая информация за смену оператора.

Посмотреть, как выглядят отчеты оператора.





7. Закрытие смены оператора.

Когда оператор заканчивает работу, то он обязан закрыть смену и сдать наработанную сумму. Оператор может сверить имеющуюся у него сумму по текущему отчету, а потом закрыть смену, выбрав меню "Оператор\Закрыть смену". При этом, если надо, можно ввести комментарий по смене, который позже прочтет менеджер.

Посмотреть, как выглядят окно закрытия смены.
