Работа с GameClass

Первый запуск

Инсталляция системы автоматически создает 2 пользователя: manager(управляющий клубом) и operator (оператор клуба). Пароли у обоих пользователей по умолчанию пустые. Настоятельно рекомендуем установить сложный пароль для пользователя manager!!!

Работа с программой начинается с подключения. Необходимо ввести имя и пароль пользователя и нажать кнопку "Войти". При этом происходит подключение к базе данных, и начинается работа с программой. По умолчанию GameClass подключается к локально-установленной базе данных MSDE2000 (т.е. к IP-адресу 127.0.0.1). Если Вы подключаетесь удаленно, то необходимо ввести IP-адрес сервера, на котором установлен MSDE-Сервер, использующийся GameClass'ом в вашем клубе. При подключении через Internet необходимо выполненение следующих условий: открыт TCP порт номер 1433, а компьютер, к которому вы подключаетесь, имеет реальный IP-адрес, и он вам известен.

В системе одновременно могут работать несколько пользователей. Например, оператор со своего рабочего места (работа с клиентами) и менеджер со своего личного кабинета (мониторинг действий оператора и просмотр отчетности). Работа одновременно двух операторов не рекомендуется.
