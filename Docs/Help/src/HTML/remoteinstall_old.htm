Удаленная инсталляция и обновление клиентов GameClass

GameClass - это система, состоящая из сервера и большого числа клиентских частей, соответствующих количеству компьютеров в вашем компьютерном салоне. На каждый клиентский компьютер нужно устанавливать клиентскую часть GameClass. У этого способа есть много недостатков - отнимается время системного администратора (клиентских машин много), а если на них сейчас работают посетители? Приходится ждать, либо терять прибыль, освобождая компьютеры. Или, допустим, вы добавляете новый компьютер. Или переходите на новую версию GameClass - и вот тут самое сложное. Компьютеры могут быть заняты посетителями. В общем, проблем хватает. Но теперь в GameClass есть уникальная система автоматический удаленной инсталляции и автоматического обновления клиентских частей GameClass! Что это означает на практике? Что вам больше не надо вручную устанавливать клиента GameClass на каждый компьютер в вашем клубе. Все это за вас сделает сам GameClass! Скорость развертывания на весь клуб составляет всего несколько секунд! Теперь работать с системой GameClass будет еще легче!

Что конкретно будет делаться автоматически?
1. Если на клиентском компьютере не установлен клиент GameClass, то он будет автоматически установлен.
2. Если на клиентском компьютере установлен клиент меньшей версии, чем сервер, то он будет автоматически обновлен до новой версии.

Если в настройках GameClass стоит галочка "Автоматически инсталлировать/обновлять клиентские части", то система каждые 5 минут будет проверять клиентские части и при необходимости устанавливать или обновлять их до новых версий. Также можно выполнить принудительную инсталляцию клиентской части, не дожидаясь пока система сама определит необходимость этого действия. Для этого нужно выделить компьютеры (или один компьютер) и выбрать меню "Действия/Дополнительные функции/Инсталляция(обновление) клиентской части".

Вопросы:
1. Какие настройки нужно сделать? Ввести имя/пароль администратора Windows (подразумевается, что это имя создано на каждом компьютере в вашем компьютерном салоне или же в случае если у вас доменная система, то вводите пароль администратора домена).
2. Как теперь происходит переход всего клуба на новую версию GameClass? Просто ставите новый GameClass сервер, а дальше все происходит автоматически.
3. Что происходит при добавлении нового компьютера? В настройках GameClass просто добавьте новый компьютер, вписав его IP в список компьютеров. Дальше все произойдет автоматически (если на новом компьютере такой же пароль администратора, который вы вводили в настройках GameClass - смотрите вопрос номер 1).
4. У меня Windows 98 и удаленная инсталляция не работает. Что делать? К сожалению, удаленная инсталляция работает только на компьютерах с операционной системой Windows XP/2000/2003. Переходите на эти операционные системы!

Важно! По умолчанию политика безопасности Microsoft настроена так, что Administrator не может подключаться к системе из сети, если для него установлен пустой пароль. Чтобы удаленная инсталляция клиентов GameClass работала корректно, необходимо задать для пользователя Administrator непустой пароль - это самое правильное решение. Если же по каким-либо причинам Вы не хотите устанавливать непустой пароль для администраторов, тогда нужно на каждом клиентском компьютере сменить политику безопасности: настройку \\Control Panel\\Administrative Tools\\Local Security Policy\\Local Policies\\Security Options\\Accounts: Limit local account use of blank password to console logon only надо установить в Disabled
