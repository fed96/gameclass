Изменения в программе GameClass
---------------------------------------------
[3.85]

Добавлены следующие возможности:
- Возможность установки GameClass на именованные инстанции (named instance)
  SQL-сервера 
- Новый редактор отчетов: сохранение параметров колонок, возможность создания
  собственных отчетов для операторов
- Новые действия для группы компьютеров: доплата, завершение работы.
- Улучшение очистки статистики: режимы, выбор таблиц для очистки и фильтр по
  времени. 
- Создание ограниченных сроком действия учетных записей с печатью пароля на
  чеке (функция оператора "Оплатить новую учетную запись")  
- Редактирование и переасчет накопленных сумм с использованием временного
  фильтра 
- Улучшение функции снимать задачи: возможность редактирования списка задач
  пользователем manager, режимы снятия задач 
- Срытие визуализации сеансов, когда не используется бронирование
- Добавление бесплатного времени
- Сдача с отдельной оплаты трафика
- Снятие задач перед началом сеанса
- Функция доплаты для клиентов по учетным записям
- Обновлен дизайн клиентской части


Исправлены ошибки:
- При смене тарифа на лету стоимость всего трафика рассчитывается по
  последнему тарифу 
- В шаблон процессов можно добавлять один и тот же процесс несколько раз 
- При печати возможно восстановление завершенной сессии 
- При добавлении нового тарифа возникает ошибка "Procedure 'TarifsAdd' expect
  parameter '@BytesInMB', which was not supplied" 


[3.84]
Добавлены следующие возможности:

- Функция менеджера "Забрать часть денег" позволяет учитывать изьятие
  владельцем клуба денег в любое время, а не только в момент закрытия смены.
- Учет в кассе денег за печать с машины оператора выполнен в виде
  автоматически вызываемой услуги "Распечатка с машины оператора". Для работы
  функции необходимо на закладке "Настройки" настроек системы заполнить поле
  "IP-адрес" и установить галочку "учитывать печать как услугу" в разделе
  "Машина оператора". 
- Трафик, использованный с машины оператора, отображается в текущем отчете
  выполнен в виде автоматически вызываемой услуги "Распечатка с машины
  оператора". Для работы функции необходимо на закладке "Настройки" настроек
  системы заполнить поле "IP-адрес" и установить галочку "считать трафик" в
  разделе "Машина оператора". 
- Запуск скриптов на клиентских машинах по событиям: запуск и остановка
  сервиса, запуск визуальной части клиентского приложения, блокировка и
  разблокировка, включение и выключение интернета, после завершения сеанса,
  потеря связи с сервером, осталось N минут до конца сеанса, изменение
  состояния клиентской машины, смена тарифа.
  Если установлена галочка "Синхронизировать с файлом на сервере", файлы
  скриптов на всех клиентских машинах автоматически синхронизируются с
  аналогичным файлом на сервере.
- Запуск скриптов на операторской машине по событиям: подключение и
  отключение, закрытие смены, услуга, изьятие части денег.
- Возможнось сохранять все данные сеанса на клиентской машине и
  восстанавливать их после перезагрузки при выключенном сервере.
- При учете трафика через плагин можно ограничивать доступный трафик, что
  позволяет доводить сессию до конца при отключенном сервере. Функция работает
  с модулем TrafficInspector Plug-in начиная с версии 2.0.
- Улучшен дизайн закладки "Трафик", добавлены способы тарификации трафика:
  только входящий, сумма входящего и исходящего, максимальное значение, а
  также опции управления работой плагина: ограничивать доступный трафик,
  ограничивать скорость, устанавливать группу.
- Изменение шрифта таблиц.
- Корректное снятие игры Lineage2 при установленном RunPadShell
- При авторизации в GameClass для  RunPadShell 6.0 запускается VIP сессия с
  таким же именем и паролем 
- Выключение монитора при включенной постоянной авторизации: Если
  активированиа опция "Выключать монитор в состоянии авторизации при простое"
  на закладке "Учетные записи", а также опция "Выключать монитор" в
  блокировках на закладке "GC3 клиент", то монитор гаснет и в состоянии
  авторизации через одну минуту после последнего нажатия клавиши на клавиатуре. 
  Включение монитора происходит при нажатии любой клавиши, если активное окно
  клиента GameClass или при нажатии Ctrl+Alt+Shif+I в любом случае. 
- В учетные записи добавлена опция "Применять скидки для тарифных пакетов"

Исправлены ошибки:
- Уменьшена высокая загрузка процессора при работе серверной части.
- При выключенном сервере после завершения сеанса не работает снятие задач. 
- На медленных компьютерах иногда происходит сбой из-за запуска двух копий.
  визуальной части клиентского приложения.
- Право на "Главный отчет" устарело.
- В текущем отчете не отображается сумма, списанная с карт клиентов.
- Опция "После завершения сеанса через ХХ сек" при отключении оператора для
  свободных машин срабатывает несколько раз.
- Перезагрузка либо выключение машины, если связь с сервером потеряна.
- Утечка памяти в процессе GCServer.exe
- При установке на чистую базу выдает ошибку.
- При установке на старые версии не добавляются процессы в шаблон процессов.
- Отсутствуют записи об окончании сеансов в консоли для менеджера.
- Нельзя изменить путь к файлу top.html в настройках клиента.
- Не работает автоматическое обновление.
- Возврат мелких сумм сдачи при включенном запрете сдачи.
- При вводе дробных значений стоимости страницы на закладке "Принтер", неверно
  переносится стоимость в закладку "Услуги". 
- Исправлена ошибка при завершении сеанса в RunPadShell "Access violation in
  module wb.exe..." 


[3.83]
Добавлены следующие возможности:

- Отмена брони: 
  В окне "Начать работу" на забронированной сеансе (сиреневая полоска) правым
  кликом мыши вызывается контекстное меню "Отменить", выбор которого приводит
  к отмене брони. 
- Небольшие изменения в дизайне серверного приложения: 
  двойной клик мыши на компьютере с запущенным сеансом вызывает доплату; 
  в "Панель инструментов" добавлена кнопка Reset;
  улучшен дизайн закладки "настройки" в "Настройках системы";
  настройки модуля контроля трафика перенесены в закладку "Трафик".
- Автоматическое обновление настроек:
  Изменения настроек под менеджером сразу отображаются в параллельно
  запущенной копии GameClass под другим пользователем.
- Учетные записи клиентов работают с фотографиями: поддреживаются форматы JPG,
  GIF и BMP.
- Новая опция в "Учетных записях": 
  Блокировать скидки, если баланс отрицательный.
- При инсталляции GameClass можно задавать путь к директории, в которую он
  будет устанавливаться
- При запрете бронирования окно "Начать работу" работает в уменьшенном виде,
  подходящем для видеорежима 800*600: 
  бронирование можно запретить на закладке "Настройки" опция "Запретить
  бронирование"  
- Утилита "Backup and Restore database" может восстанавливать бекап базы
  данных GameClass на "свежеустановленный" сервер MSDE  без предварительной
  инсталляции GameClass

Исправлены ошибки:
- Неправильное округление по времени при значениях больше 60 секунд
- Неправильный расчет сдачи для пакетных тарифов
- Не работает опция "Блокирование компьютера при неверной авторизации > 2 раз"
- Не работает блокировка на медленных машинах при старте клиента GameClass при
  установленном RunPad Shell
- Правый клик мышью на границе настраиваемых окон клиента вызывает меню
  InternetExplorer
- При вводе некоторых паролей аварийной разблокировки в "Настройках системы"
  возникает сообщение об ошибке
- При запуске сессии с незакрепленным тарифом попадающим на границу суток
  сессия исчезает через несколько секунд исчезает 
- При ограничение скорости трафика для plugin'ов реальная скорость в 8 раз
  меньше
- Отсутствовало ограничение на длину вводимого текста в некоторых поля
  редактирования клиена GameClass, что могло приводить к снятию задачи


[3.82]
Добавлены следующие возможности:

- Запуск внешних программ при старте/стопе сеанса: 
  При всех операциях с сеансами можно запускать внешний файл, которому в
  параметрах передается совершаемое действие, IP-адрес, название тарифа, имя
  клиента и доступность Интернета. 
- Отслеживание статистики популярности игр и программ:
  Клиентская часть отслеживает запуск и завершение всех процессов на
  клиентском компьютере, соответствующую статистику можно увидеть в новом
  отчете "По процессам"
- Привязка нескольких закладок Runpad к одному тарифу:
  В настройках добавлена новая закладка "RunPad", в которой можно для каждого
  тарифа отдельно настроить список доступных закладок.
- Ограничение скорости трафика в плагинах для прокси-серверов TI и UG:
  Для модулей контроля трафика под Windows (Traffic Inspector и UserGate 2.8)
  в настройке тарифов у тарифов добавлены два новых поля: 
    "Огр. скор. KB" (по умолчанию 0) - Ограничение скорости пользователя в
                                       КБайтах 
    "Группа" (по умолчанию "")       - Группа UserGate или TrafficInspector,
                                       в которую будет помещен пользователь
  которые позволяют устанавливать ограничение скорости, группу или и то и
  другое для каждого тарифа отдельно.
- Интеграция GCSecurityMonitor прямо в GameClass:
  При выборе закладки отчеты в одном случае из четырех GameClass проверяет
  целостность и безопасность базы данных:
    наличие стертых записей в базе данных,
    наличие записей с обнуленной датой,
    инсталляцию утилиты GCRemover,
    наличие администраторских и дополнительных прав у пользователей, не
    состоящими в группе менеджеров,
    устанавливает случайны пароль встроенному администратору "sa"
- "Незакрепленный" пакетный тариф, т.е. пакет, который мог бы начаться в любое
  время 
- Улучшение модуля контроля оборудования: контроль оборудования теперь
  осуществляется при помощи WMI
- Отчет "По текущему оборудов" выдает текущее состояние оборудования в зале
- Звуковое сопровождение при штрафах
- Альтернативное визуальное предупреждение во всех режимах (мерцающий метод),
  не использующий переключение десктопов, которое вызывает
  сворачиваниенекоторых игр
- При инсталляции GameClass регистрируется во встроенном в XP SP2 фaайерворле
  как разрешенное приложение

Исправлены ошибки:

- Некорректное плавающее окно после перезагрузки при тарифе "Ремонт"
- Пока пользователь не кликнет на иконку в трее в таскбаре остается окно gccl
- В режиме постоплаты на сервере не отображается отработанное время в столбце "Время"
- В окне клиентской части работает контекстное меню, при помощи которого
  пользователь может работать с файлами на заблокированном компьютере  
- Возврат денег за сеанс по предоплате: 
  Сеанс запускается с временем на 1 минуту меньше, чем должно быть. По

  времени увеличивается на 0.1 
- Неправильный перерасчет времени при доплате после применения штрафа

[3.80]

Внимание!!!
Начиная с версии 3.80, GameClass не будет поддерживать операционные системы
Windows 95/98/Me. Серверная часть 3.80 тестировалась на Windows 2000/XP. В
этих операционных системах гарантируется наиболее стабильная работа программы. 

Клиентская часть версии 3.80 полностью сделана заново и по новой технологии.
Теперь клиентская часть состоит из двух компонент. Первая - сервис (служба
Windows NT). Вторая - видимая часть, которая будет предоставлять информацию
пользователю и получать от него команды. Такая структура клиентской части
дает большие преимущества. Самое главное - надежная связь с серверной частью.
Отсутствие возможности снять клиентскую часть посетителю и многое другое.

Переход на версию 3.80 требует обновления серверной и клиентской частей:
Серверная часть.
1. Сделайте резервную копию вашей текущей базы.
2. Выполните обновление серверной части до новой версии, выбрав пункт
   "Обновление главного управляющего компьютера". При этом СОХРАНЯЮТСЯ все
   настройки и статистика. Восстанавливать сделанную резервную копию
   необходимо только в случае отката на предыдущую версию. 
Клиентские части.
1. Деинсталлируйте клиентскую часть предыдущей версии и, если потребуется,
   перегрузите компьютер. 
2. Проинсталлируйте клиентскую часть. Теперь все файлы клиентской части
   находятся в папке 
   "C:\Program Files\GameClass3 Client\"

Добавлены следующие возможности:

- Изменен интерфейс клиентской части. Интерфейс стал настраиваемым, можно
  изменить внешний вид клиентской части, используя понравившийся skin.
  Технология скинов реализована с помощью HTML-шаблонов, что позволяет
  самостоятельно изменять их. По умолчанию шаблоны лежат на каждом клиентском
  компьютере в папке "C:\Program Files\GameClass3 Client\Skins". Вы можете
  создать свой скин и расположить его на сервере в расшаренной папке. В этом
  случае путь к скинам задается в UNC-формате (пример
  "\\server\shara\top.html") 
- Возможно задать отображение "Соглашения с посетителем" в начале сеанса.
  Соглашение можно задать и изменить таким же образом как и скины (читайте
  абзац выше) 
- Просмотр текущего состояния по балансу на клиентской части по горячим
  клавишам Ctrl+Shift+Alt+I 
- Визуальное оповещение за N минут до окончания сеанса во всех видеорежимах, в
  т.ч. и в играх 
- Визуальное оповещение за 5 минут до окончания сеанса в виде всплывающей
  подсказки из SysTray, при использовании GameClass совместно с Runpad эта
  функция будет работать только если Runpad имеет версию 4.52 или выше 
- Возможность закрытия окна GameClass на клиентской части по Esc или с помощью
  стандартного значка сворачивания
